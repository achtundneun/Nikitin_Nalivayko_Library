# Nikitin_Nalivayko_Library
# Стартовым окном проекта является окно авторизации
![image](https://user-images.githubusercontent.com/80224522/163478683-d4e4cd09-7466-4f7d-8e87-f62103d4ab34.png)
# После входа в систему пользователь попадает на главное окно, на котором ему доступен переход на окна Список читателей, Список книг и выдать книгу
![image](https://user-images.githubusercontent.com/80224522/163478832-c5bfc7f4-bab3-4097-85d3-aa3def1afaa5.png)
# Окно список читателей представляет из себя таблцу с данными из базы с возможностью их поиска и сортировки по разным критериям
![image](https://user-images.githubusercontent.com/80224522/163479672-acc6c4b3-a2a2-4003-a2c3-b8dcfd54dda1.png)
# С другими Окнами ситуация аналгичная.
# Выбрав любую запись мы, нажав кнопку Delete или Backspace можем ее удалить. Послк нажатия кнопки появляется окно, которое не даст удалить запись случайно.
![image](https://user-images.githubusercontent.com/80224522/163479791-2bb694ae-873f-428e-8917-07276fba8d2d.png)
